## About

This portfolio is built as a single-page, terminal/IDE-themed site — designed to reflect a developer's actual working environment rather than a generic template. It covers:

- **About** — profile summary and quick stats
- **Skills** — technical stack (Generative AI, Backend, Cloud, DevOps, etc.)
- **Experience** — client work at Capgemini, presented git-log style
- **Projects** — personal builds (lead-scoring agent, LinkedIn auto-publisher, job-alert engine)
- **Education & Certifications**
- **Contact**

## Tech

Built with plain **HTML/CSS/JS** — no frameworks, no build step. Fonts via JetBrains Mono. Fully static, hosted free on **GitHub Pages**.
